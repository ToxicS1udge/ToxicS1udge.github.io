Hexo常用指令

给大家解释一下几个最常用到的hexo命令含义：

hexo 常用指令
hexo g hexo generate 生成静态文件
hexo d hexo deploy 将博客部署至服务器
hexo n name hexo new 加名字 生成新博文文件
hexo clean hexo clean 清除缓存文件

某些情况下，hexo clean可以解决对网页的改动不生效的情况。
